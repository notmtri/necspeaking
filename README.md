# SpeakUp – AI-Powered English Speaking Assistant

SpeakUp is a simple Python project that uses Groq’s API (LLaMA / Mixtral models) to help users practice English speaking and get AI-generated responses.

---

##  Features
- Chat with an AI using Groq API  
- Secure API key handling with `.env`  
- Easy to expand into a web app, Discord bot, or GUI  
- Clean folder structure for solo development  

---

##  Project Structure
SpeakUp-Project/
├── main.py
├── .env # contains API keys (not uploaded to GitHub)
